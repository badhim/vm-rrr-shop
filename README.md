# README

* Prerequisites

Ensure you have the following installed:

    Ruby (version >= 3.0)
    Rails (version >= 8.0)
    Node.js
    PostgreSQL

* Install dependencies

    bundle insltall
    npm install

* Set up database

    rails db:create
    rails db:migrate

* Seed the database (optional)

    rails db:seed

* Start the server

    rails s
